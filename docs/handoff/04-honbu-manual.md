# 本部オペレーションマニュアル（管理画面 /admin の使い方）

最終更新: 2026-07-20

この資料は、本部担当者のためのものです。保存された写真の確認と、店舗・担当者・写真枠の管理のしかたを説明します。

## 0. 一番大事な前提

**日常のデータ管理（担当者・枠・写真の管理、店舗情報の編集）は、すべてこの管理画面でできます。**
この画面にない操作（店舗の新規追加、ログインPINの設定など）は技術担当の仕事です。データの金庫（Supabase）を自分で直接開けようとしないでください。

## 1. 入り方と出方

- パソコンやiPadのブラウザで https://dog-sns-prototype.vercel.app/admin を開きます。
- 本部PIN（暗証番号）を入力すると管理画面が開きます。
- 本部PINは1つだけです。本部の管理を任された人以外に教えないでください。紙に書いてレジ横に貼る、といった扱いは厳禁です。
- 終わったらログアウトします。共用の端末では特に忘れずに。

## 2. 画面の全体像

管理画面には3つのタブがあります。

| タブ | できること |
|------|-----------|
| 写真 | 保存された完成写真の確認、メモ、非表示、完全削除 |
| 店舗 | 店舗の追加・編集・停止と、各店舗の担当者の管理 |
| 枠 | 店舗ごとの写真枠（最大3枚）の登録・差し替え・並び替え・削除 |

## 3. 写真タブ

店舗でお客様のSNS掲載OKをもらって保存された完成写真が、新しい順に並びます。

- **一覧**: 店舗や状態で絞り込めます。一度に表示されるのは160件までで、それより多いときは画面の案内に従って絞り込んでください。
- **写真を選ぶと詳細が開き**、次の操作ができます。
  - **一言メモ**: 「柴犬のモモちゃん、常連さん」など、あとで見返すためのメモを残せます。
  - **確認状態**: 「未確認・投稿候補・保留・使用しない」の4段階の印を付けられます。SNSに載せる写真を選ぶ作業用の印です。どう使い分けるかは本部で決めてください（システムは印を覚えるだけで、何かを自動でしたりはしません）。
  - **非表示にする**: 日常の一覧から見えなくします。写真は消えておらず、いつでも復帰できます。
  - **完全に削除する**: 写真をデータごと消します。**元に戻せません。**

### 非表示と完全削除の使い分け（大事）

- ふだんは**非表示**を使ってください。消したように見えて、戻せます。
- **完全削除は、お客様から「消してほしい」と申し入れがあった場合の最後の手段**です。確認のダイアログが出ますが、実行すると写真もデータも消え、誰にも戻せません。
- お客様からの削除申し入れには、この完全削除で必ず応じてください。これは写真を預かる側の義務です。

## 4. 店舗タブ

店舗の一覧が「店名・店舗コード・状態」のカードで並びます。

- **店舗を選ぶと**、右側で店舗の情報（店舗名、表示名、並び順、有効/無効、メモ）を編集できます。
- **この画面でできないこと**: 新しい店舗の追加と、店舗コード・ログインPINの設定・変更は、ここではできません。必要なときは技術担当に頼んでください（間違い防止のため、わざと管理画面に置いていない機能です）。
- **店舗の停止**: 閉店・休止した店舗は削除ではなく停止（無効）にします。停止中の店舗はログインできなくなります。
- **店舗の削除**: 間違い防止のため、**「停止中」かつ「保存写真が0件」の店舗しか削除できません**。営業中の店舗をうっかり消してしまう事故は起きない作りです。

### 担当者の管理

各店舗のカードの下に、その店舗の担当者が並んでいます。

- **「＋追加」** で新しい担当者を登録します。入力するのは担当者名だけです。
- 担当者を選ぶと、名前の変更や停止ができます。
- 停止した担当者は、店舗画面の「担当者を選ぶ」に出なくなります。辞めた人は停止してください。

## 5. 枠タブ

店舗を選ぶと、その店舗の写真枠が最大3枚のサムネイル（小さな見本画像）で表示されます。

- **枠の追加**: 白紙のスロット（空き枠）を押して、枠の画像ファイルを登録します。
- **枠の差し替え**: 既存のサムネイルを押すと、画像の差し替えと、日付が写真のどこに入るかの位置調整ができます。
- **並び替え**: サムネイルをドラッグして順番を変えられます。**一番左の枠が「標準枠」**になり、店舗画面で最初に選ばれています。
- **削除**: 枠は消してもかまいません。過去に保存した写真には枠が焼き付いているので、枠を消しても写真は変わりません。

### 枠の画像ファイルの決まり

- **正方形であること**（縦横が同じ長さ）。正方形でないファイルは登録できません。
- 形式は**透明部分のあるPNG、またはSVG**。JPEGやWebPは透明（写真が見える窓の部分）を扱えないため、枠には使えません。
- PNGは**1080ピクセル以上**であること。小さいものは登録できません。ファイルの大きさは**2MBまで**。
- 新しい枠のデザインを作る・外注するときの詳しい決まりは、技術担当に `docs/frame-image-guide.md` を見てもらってください。

## 6. やってはいけないこと

- 本部PINを本部担当以外に教えること。
- 完全削除を「整理」目的で日常的に使うこと（非表示で足ります）。
- データの金庫（Supabase）や設計図（GitHub）を直接触ること。これらは技術担当の領分です。
- お客様に「SNSに自動で載ります」と説明すること（自動投稿の機能はありません）。
