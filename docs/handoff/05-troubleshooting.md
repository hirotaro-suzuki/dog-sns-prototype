# 困ったときの手引き（店舗・本部の担当者向け）

最終更新: 2026-07-20

トラブルが起きたときに、慌てずに確認する順番と、誰に相談するかをまとめた資料です。

## 0. 最初に覚えておくこと

- **保存済みの写真は、iPadが壊れても消えません。** 写真はクラウドの金庫（資料2参照）にあります。iPadのトラブルで写真が失われることはないので、落ち着いて対処してください。
- 多くのトラブルは「ページの再読み込み」か「iPadの再起動」で直ります。まずこの2つを試してください。

## 1. 症状別の確認手順

### 画面が開かない・真っ白になる

1. iPadがWi-Fiにつながっているか確認する（他のサイトが開くか試す）。
2. ページを再読み込みする。直らなければiPadを再起動する。
3. 別の端末（スマホなど）で同じURLを開いてみる。別の端末でも開かない場合はシステム側の問題なので、技術担当へ連絡する。

### ログインできない

1. 店舗コードとPINの打ち間違いがないか、ゆっくり入力し直す。
2. それでも入れない場合、店舗が「停止」になっていないか本部に確認する。
3. PINが分からなくなった場合は、本部経由で技術担当に再設定を頼む（画面でPINを確認する方法はありません）。

### カメラが映らない

1. 画面を再読み込みする。
2. iPadの設定でSafariのカメラ利用が許可されているか確認する（設定 → アプリ → Safari → カメラ）。
3. 直らなければiPadを再起動する。

### 保存ボタンでエラーが出る

1. Wi-Fiがつながっているか確認して、もう一度保存を押す。
2. 何度やってもエラーになる場合は、時間をおいて試す。急ぎでなければ、その回はお客様に印刷だけ渡して対応する。
3. 続くようなら技術担当へ連絡する（金庫側の問題の可能性があります）。

### 印刷できない

1. プリンターの電源、用紙、インクを確認する。
2. iPadとプリンターが**同じWi-Fi**につながっているか確認する。
3. 直らなければ、iPadとプリンターの両方を再起動する。
4. 印刷だけの問題なら、写真の保存はできます。お客様には後日渡すなどの対応も検討してください。

### 写真がぼやける・暗い

- レンズを柔らかい布で拭く。明るい場所で、わんちゃんに少し近づいて撮る。逆光を避ける。
- それでも全体的におかしい場合は技術担当へ相談する。

### 本部の管理画面（/admin）が開かない・PINが通らない

1. URLの最後に `/admin` が付いているか確認する。
2. 本部PINの打ち間違いを確認する。
3. PINが分からなくなった場合は、技術担当に再設定を頼む。

## 2. 技術担当に相談するとき伝えること

次の3点を伝えると、解決がとても早くなります。

1. **いつ**: 日時。いつから起きているか。
2. **どの画面で**: ログイン画面、撮影画面、編集画面、/admin など。
3. **何をしたら、何が出たか**: 押したボタンと、出たメッセージ。**エラーメッセージは、iPadのスクリーンショット（電源ボタン＋音量上ボタンを同時押し）か、スマホで画面を撮って送るのが確実です。**

## 3. 緊急のとき（写真を見られたくない事態など）

万一、写真の流出が疑われる、システムが悪用されている、といった緊急事態のときの止め方です。

- **1店舗だけ止める**: 本部担当者が /admin の店舗タブで、その店舗を「停止」にする。以後その店舗はログインできなくなります。
- **システム全体を止める**: 技術担当に連絡し、公開そのものを一時停止してもらう（Vercel側の操作。技術担当は資料7参照）。
- 判断に迷ったら、まず止めてから相談してください。止めても保存済みの写真とデータは消えません。

## 4. 誰に相談するかの早見表

| 困りごと | 相談先 |
|----------|--------|
| 操作がわからない | まず資料3・4のマニュアル、次に本部担当者 |
| 担当者の追加、枠の変更、写真の非表示・削除 | 本部担当者（/adminでできます） |
| 店舗の追加、PINの再設定、エラーが直らない | 技術担当 |
| 契約・支払い、アカウントの持ち主に関わること | オーナー |
