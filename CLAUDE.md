# CLAUDE.md

このファイルはClaude Codeが起動時に自動で読み込む入口です。ユーザーが開始文を貼り忘れても、最低限このルールで作業を始められるようにするためのものです。詳しい内容は `docs/START_HERE.md` と `docs/DOC_INDEX.md` を参照してください。

## 必ず守ること

- 作業対象は GitHub `hirotaro-suzuki/dog-sns-prototype` の `main`。作業環境はこの GitHub Codespaces を正とする。
- ローカルPC、Dropbox、手元フォルダ、古い複製リポジトリは参照しない。
- Claude Codeは、自身のツールを用いて直接ファイルを編集する。
- 実装、ビルド確認、コミット、pushは、Claude Code自身がターミナルコマンドを実行して行う。
- トークン節約のため、いきなりプロジェクト内を広範囲に探索しない。まず `docs/DOC_INDEX.md` の地図に従って、必要な文書だけを読む。
- 実装前に、目的・OK条件・確認方法・更新する文書を短く整理し、ユーザーの許可を得てから編集・実行する。
- ユーザーの指示や質問（アイデア・思いつき・軽い提案を含む）に対して、すぐに実装に入らない。まず自分の意見・懸念・気づいた点を言葉で返し、合意してから編集・実行する。
- エラーで詰まった場合は、ローカルやDropboxへ逃げず、何ができて何ができないかをユーザーへ説明して止まる。
